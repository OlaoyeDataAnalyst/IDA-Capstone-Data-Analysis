## 📁 Projects

### 📊 Project 1 — Retail Sales Performance Analysis
- **Domain:** E-Commerce / Retail
- **Company:** ShopEase Nigeria
- **Dataset:** 79,931 orders | 2021–2023
- **Tools:** Python | SQL | Excel | Power BI
- **Key Finding:** Lagos leads in revenue; Smartphones 
  are the top product; near-zero repeat purchase rate 
  identified requiring urgent loyalty programme
- 📂 [View Project](./Project1_ShopEase)

---

### 🏥 Project 2 — Hospital Patient Flow Analysis
- **Domain:** Healthcare
- **Company:** Lagos General Hospital
- **Dataset:** 59,950 patient records | 2021–2023
- **Tools:** Python | SQL | Excel | Power BI
- **Key Finding:** ₦6 billion revenue collection gap; 
  Pediatrics is most burdened department; peak 
  admissions at Friday 5PM
- 📂 [View Project](./Project2_Hospital)

---

### 💰 Project 5 — Personal Finance Analysis
- **Domain:** Consumer Finance / Fintech
- **Company:** FinTrack Nigeria
- **Dataset:** 74,922 transactions | 5,000 users | 2021–2023
- **Tools:** Python | SQL | Excel | Power BI
- **Key Finding:** 99.6% of users overspend monthly; 
  overall savings rate is 16.80%; 1,655 users have 
  never saved
- 📂 [View Project](./Project5_FinTrack)
